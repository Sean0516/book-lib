# 基础8模块 - 总目录

这份目录是给基础面试学习做导航用的，你后面可以直接从这里进入每个模块。

## 一、模块清单

1. [基础模块1 - Java基础与集合](/fundamentals/基础模块1-Java基础与集合-标准答案库)
   重点：Java 语言基础、集合体系、泛型、反射、注解、序列化

2. [基础模块2 - JVM基础](/fundamentals/基础模块2-JVM基础-标准答案库)
   重点：运行时数据区、类加载、GC、Full GC、OOM、调优与排障

3. [基础模块3 - 并发基础](/fundamentals/基础模块3-并发基础-标准答案库)
   重点：JMM、volatile、synchronized、AQS、线程池、ThreadLocal、死锁

4. [基础模块4 - 计算机基础](/fundamentals/基础模块4-计算机基础-标准答案库)
   重点：操作系统、IO、TCP/HTTP/HTTPS、Linux 排障、网络抖动定位

5. [基础模块5 - 数据库基础](/fundamentals/基础模块5-数据库基础-标准答案库)
   重点：ACID、隔离级别、索引、MVCC、锁、慢 SQL、主从复制、分库分表

6. [基础模块6 - 缓存与消息基础](/fundamentals/基础模块6-缓存与消息基础-标准答案库)
   重点：Redis、缓存一致性、缓存问题治理、Kafka/MQ、消息可靠性与积压

7. [基础模块7 - Spring基础](/fundamentals/基础模块7-Spring基础-标准答案库)
   重点：IOC、AOP、事务、循环依赖、MVC、自动配置、启动慢排查

8. [基础模块8 - 算法与数据结构基础](/fundamentals/基础模块8-算法与数据结构基础-标准答案库)
   重点：复杂度、链表树图、二分、双指针、滑动窗口、回溯、DP、LRU

## 二、推荐学习顺序

如果你想系统复习，建议按这条顺序走：

1. 基础模块1 - Java基础与集合
2. 基础模块2 - JVM基础
3. 基础模块3 - 并发基础
4. 基础模块5 - 数据库基础
5. 基础模块6 - 缓存与消息基础
6. 基础模块7 - Spring基础
7. 基础模块4 - 计算机基础
8. 基础模块8 - 算法与数据结构基础

这样安排的原因是：
- 前三块先把 Java 后端最核心的语言、运行时和并发地基打稳
- 中间三块是最常见的后端面试主战场
- 计算机基础适合在中后段和前面知识互相映射
- 算法最后单独冲刺，效率通常更高

## 三、每个模块怎么用

每个模块都统一包含这些部分：

- `核心结论`：先快速记结论
- `标准答案`：自然口述版，适合面试回答
- `追问点答案`：帮助你往下深挖
- `原理补充`：补底层机制
- `架构扩展`：帮助你从知识点走向工程理解
- `工程指标`：训练你带指标回答
- `证据链`：训练你排障和论证思路
- `低风险优化路径`：训练你工程落地的优先级
- `易错点`：方便你查漏补缺

## 四、推荐复习节奏

如果你每天能拿出 2 小时，比较稳的节奏是：

1. 40 分钟看一个模块的 `核心结论 + 标准答案`
2. 40 分钟重点看 `追问点答案 + 原理补充`
3. 20 分钟只看 `架构扩展 + 工程指标`
4. 20 分钟自己口述 3 到 5 题
